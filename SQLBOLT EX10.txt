Ques.1: Find the longest time that an employee has been at the studio.
Query : SELECT MAX(years_employed)
	FROM employees;

Ques.2: For each role, find the average number of years employed by 	employees in that role.
Query: SELECT role, AVG(years_employed)
	FROM employees
	GROUP BY role;

Ques.3: Find the total number of employee years worked in each building.
Query : SELECT building, SUM(years_employed)
	FROM employees
	GROUP BY building;
