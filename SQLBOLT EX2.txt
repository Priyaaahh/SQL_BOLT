Ques.1: Find the movie with a row id of 6.
Query : select * from movies where id = 6;

Ques.2: Find the movies released in the years between 2000 and 2010.
Query : select * from movies where year between 2000 and 2010;
 
Ques.3: Find the movies not released in the years between 2000 and 2010.
Query : select * from movies where year not between 2000 and 2010;

Ques.4: Find the first 5 Pixar movies and their release year.
Query : select title, year from movies order by year asc limit 5;
